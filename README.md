log-monitoring-tool

監視・運用業務を想定したログ解析ツールです。
Pythonでログを生成し、エラーや警告の傾向を自動分析できます。

概要

監視業務で扱うようなログを想定し、
以下の情報を自動で集計します。
 • エラー総数
 • 警告総数
 • デバイス別エラー回数
 • 最もエラーが多い機器
 • 時間帯別エラー発生傾向

手動でログを確認するのではなく、
Pythonで効率的に状況把握する練習用ツールとして作成しました。

使用技術
 • Python
 • 辞書による集計処理
 • 文字列解析（split）
 • ソート処理
 • ログデータ分析

ファイル構成

random_log.py
→ ランダムな監視ログを生成

analyze.py
→ ログを解析しエラー傾向を出力

使用手順

１.py rondam_log.py
２.py analyze.py

作成目的

未経験からインフラ・運用監視分野への転職を想定し、
ログ確認・障害傾向分析の基礎スキル習得のため作成しました。

監視業務における
「ログ確認」「障害傾向の把握」「簡易自動化」
を意識したツールです。
